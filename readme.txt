Apizee Contact is a chat, audio and video conferencing solution, allowing you to communicate directly with your visitors.

== Description ==

Apizee Contact is a real-time web communication solution helping you deliver better customer service. Live chat, audio and videoconferencing services enable you to offer better support and advice to visitors. You can interact with your website visitors, display personalized messages at the right time and promote your products. IzeeChat helps you increase sales and customer satisfaction.
Apizee Contact can be customized to fit your unique branding and style.
The solution is based on a WebRTC platform and does not require users to download, register or install plugins to use the service.

More Information : https://www.apizee.com


== Installation ==

This section describes how to install the plugin and get it working.

1. Extract `izeechat` folder from a zip file
2. Upload `izeechat` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Create/Signin to your Apizee account and copy your site key
5. Paste site key in your module configuration
6. Nice, chat module can now loaded on your website

== Frequently asked questions ==

= How do I activate my Apizee Contact plugin? =
You have to create an Apizee account if you want to use this plugin. Copy-paste your site key in your module configuration.

= How get my site key? =
When you're connected on Apizee platform, create a new website in configuration and then you will be able to get the site key.

= My plugin doesn't work! =
Please contact Apizee support (support@apizee.com)
